Given an array nums, write a function to move all 0's to the end of it while maintaining the relative order of the non-zero elements.
Учитывая массив nums, напишите функцию, чтобы переместить все 0 в конец, сохраняя относительный порядок ненулевых элементов.

Example:

Input: [0,1,0,3,12]
Output: [1,3,12,0,0]

Note:

Вы должны сделать это на месте, не создавая копию массива.
     Минимизируйте общее количество операций.
