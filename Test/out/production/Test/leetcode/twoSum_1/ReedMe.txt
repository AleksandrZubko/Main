Получив массив целых чисел, верните индексы двух чисел так, чтобы они суммировались с определенной целью.
Вы можете предположить, что каждый вход будет иметь только одно решение, и вы не можете использовать один и тот же элемент дважды.

Example:

Given nums = [2, 7, 11, 15], target = 9,

Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1].
