
Вам даны два непустых связанных списка, представляющих два неотрицательных целых числа. Цифры хранятся в обратном порядке,
и каждый из их узлов содержит одну цифру. Добавьте два числа и верните их в виде связанного списка.

Вы можете предположить, что два числа не содержат начального нуля, кроме самого числа 0.


Example:

Input: (2 -> 4 -> 3) + (5 -> 6 -> 4)
Output: 7 -> 0 -> 8
Explanation: 342 + 465 = 807.
